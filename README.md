# Azure-Sentinel-XDR-Labs

A collection of hands-on mini projects documenting my learning journey with Microsoft Sentinel and Microsoft Defender XDR, focused on building practical SOC analyst skills.

---

## 📁 Project Structure

| Folder | Title | Focus |
|--------|-------|-------|
| [`mini-project-01`](./mini-project-01/) | SOC Foundations with Sentinel & Defender | Environment setup, log ingestion, KQL, detection rules, incident investigation |
| [`mini-project-02`](./mini-project-02/) | Email Threat Detection & Phishing Simulation | Email security policies, phishing simulation, Threat Explorer, incident report |
| [`mini-project-03`](./mini-project-03/) | Endpoint Security with Defender for Endpoint & Intune | VM onboarding, ASR policies, Atomic Red Team simulations, alert investigation |
| [`mini-project-04`](./mini-project-04/) | Identity Protection with Microsoft Entra ID | Conditional Access policies, named locations, Entra ID log ingestion into Sentinel |
